📌  RARKING 구조 설명 : 코드 작성 단계


1. 패키지 import

1) 코드의 시작 부분에서 필요한 패키지들을 import 합니다.

2) 필요하다면 pubspec.yaml에 패키지들을 추가한 후, 터미널에서 flutter pub get 명령어를 사용하여 프로젝트의 의존성을 최신 버전으로 업그레이드합니다.


2. UserRanking 클래스

1) 사용자 랭킹 정보를 저장하기 위한 클래스를 정의합니다.

2) 순위(rank), 이메일(email), 플레이타임(playtime)을 포함합니다.


3. RankingPage 클래스 (StatefulWidget)

1) 랭킹 페이지를 나타내는 StatefulWidget를 정의합니다. 랭킹 정보는 계속해서 변하기 때문에 Stateless 위젯이 아닌 StatefulWidget 위젯을 사용합니다.

2) 사용자 랭킹 정보를 서버에서 받아와 화면에 표시합니다.


4. _RankingPageState 클래스 (상태 관리)

1) RankingPage 클래스의 상태를 관리하는 _RankingPageState 클래스를 정의합니다.


5. initState 메서드

1) 위젯이 초기화될 때 서버에서 랭킹 데이터를 받아오는 userRanking 함수를 호출합니다.


6. userRanking 함수 (랭킹 데이터 받아오기)

1) 서버에서 랭킹 데이터를 받아오는 비동기 함수입니다.

2) HTTP POST를 사용하여 서버에 요청을 보내고 응답에 따라 데이터를 처리합니다.


7. updateRankingList 메서드 (데이터 업데이트)

1) 받아온 데이터로 userRankingList를 업데이트합니다.


8. parseRankingData 메서드 (랭킹 데이터 파싱)

1) 서버에서 받아온 데이터를 UserRanking 객체로 파싱합니다.


9. build 함수

1) 화면의 UI를 정의하는 함수를 작성합니다.

2) Scaffold를 사용하여 앱의 기본 구조를 정의합니다.

3) DataTable을 사용하여 랭킹 정보를 표시합니다.
