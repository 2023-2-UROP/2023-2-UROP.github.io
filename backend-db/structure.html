📌  DB 설정 단계 설명


1. accounts 테이블 생성

1) 로그인, 회원가입을 위한 account 테이블을 생성합니다.


2. playtime 테이블 생성

1) 각각의 유저의 게임 플레이 시간을 저장하기 위해 playtime 테이블을 생성합니다.

2) 이때 유저는 여러 플레이 타임 기록을 가질 수 있으므로 일대다 관계로 설계합니다.


3. 변경사항을 적용

1) 알맞은 디렉토리에서 변경사항을 적용합니다.


4. 필요한 라이브러리 설치

1) 로그인과 회원가입 등의 기능 구현에 필요한 라이브러리를 설치하고 view.py에서 사용할 수 있도록 import합니다.


5. 기능 구현

1) 로그인, 회원가입 등의 기능을 구현합니다.


6. api url 설정

1) url을 설정하고 연결하는 코드를 작성합니다.
