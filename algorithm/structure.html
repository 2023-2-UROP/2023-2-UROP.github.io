📌  알고리즘 구조 설명 : 코드 작성 단계


1. 스도쿠 배열 생성

1) 9*9 스도쿠 게임 배열을 생성합니다.

2) 스도쿠 규칙에 맞는 2차원 배열을 반환하도록 조건에 맞는 알고리즘을 작성합니다.


2. 스도쿠 정답 배열 생성

1) 규칙에 맞는 스도쿠 문제의 정답 배열을 반환하는 알고리즘을 작성합니다.

2) 스도쿠 배열을 입력으로 받고, 해결 가능 여부를 판단하여 빈 칸에 숫자를 채우는 방식으로 구현합니다.


3. 알고리즘 실행 전용 파일 생성

1) 구현한 알고리즘에 사용된 언어에 따라 해당 언어를 통한 알고리즘을 프로젝트에서 실행하기 위해 필요한 라이브러리를 불러옵니다.

2) 실행파일에서 다른 언어로 구현된 알고리즘을 테스트하고 프로젝트에 적용합니다.
