2. My 클래스

2.1 사용자 랭킹 정보를 나타내는 클래스로, 서버에서 받아오는 정보들을 표시하고 있습니다.

2.2 데이터 모델을 정의하여 서버로부터 받아온 사용자 순위 데이터를 효과적으로 다룰 수 있게끔 구조화하였습니다.

2.2 required 키워드를 통해 해당 속성이 생성자를 통해 반드시 제공되게끔 구성하였습니다. 이는 코드를 작성하면서 누락된 데이터나 부적절한 데이터가 들어오는 것을 방지하고, 타입 안정성을 확보합니다.


                
                  class My {
                    final int rank;
                    final String playtime;
                  
                    My({
                      required this.rank,
                      required this.playtime,
                    });
                  }


3. MyPage 클래스 (StatefulWidget)

3.1 랭킹 페이지를 위한 StatefulWidget입니다. 화면의 상태가 변경될 때마다(랭킹이 바뀔 때마다) 화면이 다시 그려져야 하기 때문에, StatefulWidget 위젯을 사용하였습니다.

3.2 이 위젯의 상태는 _MyPageState에서 관리됩니다.


                
                  class MyPage extends StatefulWidget {
                    const MyPage({Key? key}) : super(key: key);
                  
                    @override
                    _MyPageState createState() => _MyPageState();
                  }


4. _MyPageState 클래스 (상태 관리)

4.1 랭킹 페이지의 상태를 나타내는 함수입니다. 사용자 랭킹 목록을 초기화하고, 위젯이 초기화될 때 서버에서 랭킹 데이터를 받아오는 함수를 호출합니다.

4.2 _MyPageState 클래스는 MyPage 위젯의 상태를 관리하기 때문에, State를 상속합니다.

4.3 랭킹 페이지에서 표시할 사용자 랭킹 데이터를 담기 위한 리스트를 선언하고 초기화합니다. 처음에는 데이터가 없으므로 빈 리스트로 시작합니다.

4.4 위젯이 처음으로 화면에 나타날 때(initState 메서드), 서버에서 랭킹 데이터를 받아오는 함수를 호출하고 있습니다.


                
                  class _MyPageState extends State {
                    List myRankingList = [];
                  
                    @override
                    void initState() {
                      super.initState();
                      // 위젯이 초기화될 때 ranking 함수 호출
                      ranking();
                    }


5. ranking 함수 (랭킹 데이터 받아오기)

5.1 서버에서 랭킹 데이터를 받아오는 비동기 함수입니다.


                
                  Future ranking() async {
                    final SharedPreferences prefs = await SharedPreferences.getInstance();
                    final String? email = prefs.getString('email');

                    try {
                      final response = await http.post(
                        Uri.parse('http://ec2-54-172-150-42.compute-1.amazonaws.com:8080/users/ranking'),
                        headers: {
                          'Content-Type': 'application/json',
                        },
                        body: jsonEncode({
                          "email": email,  // 이메일에 맞는 정보를 받기 위해 이메일을 포함하여 요청
                        }),
                      );

                      if (response.statusCode == 200) {
                      final Map data = jsonDecode(response.body);
                      final List rankingData = parseRankingData(data);
                      updateRankingList(rankingData);
                      print('데이터 받아오기 성공');
                    } else {
                      print('데이터 받아오기 실패, statusCode: ${response.statusCode}');
                    }
                  } catch (error) {
                    print('에러 발생: $error');
                  }
                }

6. parseRankingData 메서드 (랭킹 데이터 파싱)

6.1 서버에서 받아온 데이터를 가공하여 My 객체의 리스트로 변환합니다.

6.2 로그인한 사용자의 플레이타임를 이용하여 순위(rank)를 부여하고, 이를 리스트에 추가하여 반환합니다. 여기서 map 함수는 리스트의 각 요소에 대해 주어진 함수를 적용하고, toList 함수는 그 결과를 리스트로 변환합니다.


                
                  List parseRankingData(Map data) {
                    List durations = (data['durations'] as List).cast();
                    int rank = 1;
                    return durations.map((duration) {
                      final My my = My(
                        rank: rank,
                        playtime: duration,
                      );
                      rank++;
                      return my;
                    }).toList();
                  }


7. updateRankingList 메서드 (데이터 업데이트)

7.1 서버에서 랭킹 데이터를 받아오는 비동기 함수입니다. 가공된 rankingData를 사용하여 userRankingList를 업데이트합니다.

7.2 setState 함수는 상태를 변경할 때 사용되며, 여기서는 새로운 랭킹 데이터로 UI를 갱신합니다.


                
                  void updateRankingList(List rankingData) {
                    setState(() {
                      myRankingList = rankingData;
                    });
                  }


8. build 메서드 (화면 렌더링)

8.1 화면을 렌더링하는 메서드입니다.

8.2 해당 코드에서는 랭킹 정보를 DataTable을 사용하여 표시합니다.


                
                  @override
                  Widget build(BuildContext context) {
                    return Scaffold(
                      appBar: AppBar(
                        title: const Text('My'),
                      ),
                      body: Center(
                        child: Column(
                          mainAxisAlignment: MainAxisAlignment.center,
                          children: [
                            const Text(
                              // 표 제목
                              '내 랭킹',
                              style: TextStyle(
                                fontSize: 17,
                                fontWeight: FontWeight.bold,
                              ),
                            ),
                            const SizedBox(height: 10),
                            DataTable(
                              // 표
                              columns: const [
                                DataColumn(label: Text('순위')),
                                DataColumn(label: Text('걸린 시간')),
                              ],
                              rows: myRankingList.map((user) {
                                return DataRow(
                                  cells: [
                                    DataCell(Text(user.rank.toString())),
                                    DataCell(Text(user.playtime)),
                                  ],
                                );
                              }).toList(),
                            ),
                          ],
                        ),
                      ),
                    );
                  }
                }
