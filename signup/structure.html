📌  SIGNUP 구조 설명 : 코드 작성 단계


1. 패키지 import

1) 코드의 시작 부분에서 필요한 패키지들을 import 합니다.

2) 필요하다면 pubspec.yaml에 패키지들을 추가한 후, 터미널에서 flutter pub get 명령어를 사용하여 프로젝트의 의존성을 최신 버전으로 업그레이드합니다.


2. SignupPage 클래스

1) 'SignupPage' 클래스가 페이지의 주요 뼈대를 담당합니다.

2) 상태가 변하지 않는 페이지 정의를 위해 StatelessWidget을 상속받습니다.


3. TextEditingController

1) 이메일, 비밀번호, 비밀번호 확인 필드의 텍스트를 관리할 세 개의 'TextEditingController'를 각각 선언합니다.


4. showResultDialog 함수 (회원가입 시도 결과 표시 함수)

1) 회원가입 결과를 팝업으로 표시하기 위한 함수를 작성합니다.

2) 서버로부터 성공 또는 실패 메시지를 전달받아 AlertDialog를 생성합니다.

3) '확인' 버튼을 포함하여 버튼 클릭 시 팝업이 닫히도록 설정합니다.


5. signup 함수 (회원가입 시도 함수)

1) 서버에 회원가입 요청을 보내는 비동기 함수를 작성합니다.

2) HTTP POST를 사용하여 서버에 이메일과 비밀번호 정보를 전송하고, 응답에 따라 성공 또는 실패 메시지를 출력합니다.

3) 'showResultDialog' 함수를 활용하여 서버 응답에 따른 회원가입 결과를 사용자에게 팝업으로 알려줍니다.


6. build 함수

1) 화면의 UI를 정의하는 함수를 작성합니다.

2) Scaffold를 사용하여 페이지의 기본 구조를 정의합니다.

3) Scaffold 위에 AppBar와 사용자로부터 입력받아야 할 항목을 담은 입력 필드, 그리고 회원가입 버튼을 배치합니다.

4) AppBar는 페이지 상단에 위치하고, 그 아래에 입력 필드들과 회원가입 버튼을 세로로 나열합니다.

5) 회원가입 버튼은 누를 때 'signup' 함수가 호출되도록 설정합니다.


7. Widget 구성

1) TextField - 사용자로부터 이메일, 비밀번호, 비밀번호 확인을 입력받는 데 사용되는 위젯입니다.

2) ElevatedButton - 회원가입 버튼을 나타내는 위젯으로, 클릭 시 'signup' 함수가 호출됩니다.
