📌  앱 빌드 방법


→  앱 빌드 예시는 Flutter 프레임워크를 사용하여 앱을 빌드 합니다. Flutter 설치 후 빌드를 진행해 주세요.

0. Flutter 설치 확인

        

    flutter doctor -v

Flutter CLI에서 위의 명령어를 통해 Flutter 설치가 올바르게 되었는지 확인 합니다.


        

    flutter pub upgrade --dry-run

위의 명령어를 통해 Flutter 버전 업그레이드가 가능합니다.



1. 프로젝트 생성 및 빌드


        

    flutter create [프로젝트 이름]
        
    

        

    flutter pub get
        
    

        
        
    flutter run

    flutter run --device-id [device_id] // 특정 기기에서 앱을 실행하고 싶을 때
        
    

        

    flutter build apk

    flutter build ios // 타겟 플랫폼 옵션
    flutter build apk --debug // 빌드 모드 지정
    flutter build apk --release //


각각의 명령어에 대한 설명입니다.

1.1 새 프로젝트를 생성하는 명령어입니다.

1.2 pubspec.yaml 파일에 나열된 의존성 패키지들을 설치하는 명령어입니다.

1.3 플러터 앱을 실행하는 명령어입니다. --divice-id 옵션을 통해 기기 id를 지정할 수 있습니다.

1.4 플러터 앱을 빌드하는 명령어입니다. 기본적으로 안드로이드 APK로 빌드 되며, 추가 옵션을 통해 타겟 플랫폼, 빌드 모드를 변경할 수 있습니다.
