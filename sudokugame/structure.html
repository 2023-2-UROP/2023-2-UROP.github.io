📌  Sudokugame 구조 설명 : 코드 작성 단계


1. 환경설정

1) 코드의 시작 부분에서 필요한 패키지들을 import 합니다.

2) 필요하다면 pubspec.yaml에 패키지들을 추가한 후, 터미널에서 flutter pub get 명령어를 사용하여 프로젝트의 의존성을 최신 버전으로 업그레이드합니다.


2. 스도쿠 게임보드 만들기(sudokuBoard)

1) 9*9 2차원 배열의 스도쿠 게임보드를 생성합니다.

2) 스도쿠 게임 보드에 숫자 업데이트 기능을 추가합니다.


3. 숫자패드 만들기(Widget buildNumberPad)

1) 게임 보드에 원하는 숫자를 입력하기 위해 1~9까지의 숫자 패드를 생성합니다.


4. 기능 버튼 만들기

1) 스도쿠에 필요한 기본적인 기능 버튼을 생성합니다.


5. 버튼 기능 추가

1) 생성한 버튼에 필요한 기능을 추가합니다.


6. 배치 밎 완성

1) 구현한 요소들을 화면에 알맞게 배치하여 완성합니다.

2) HTTP POST를 사용하여 서버에 요청을 보내고 응답에 따라 데이터를 처리합니다.
